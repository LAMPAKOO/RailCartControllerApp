# RailCartController

A Python script with both GUI and CLI interfaces for connecting to an ESP32-based stepper motor controller used in a specialized rail cart for gluing fiber-optic sensors onto rails. It provides glue gun speed and direction control, measures the cart speed using a rotary encoder mounted on the track, measures distance, and saves distance logs to a file. The script also includes control of an industrial AC motor inverter (VFD) used as the drive for the cart.
